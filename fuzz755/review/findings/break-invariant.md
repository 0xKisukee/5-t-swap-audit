**Title:**

Extra token incentive on swaps is breaking the protocol's invariant

**Description:**

Every 10 swaps, the protocol is giving the swap caller an extra token as an extra incentive to keep trading on T-Swap. However, this behavior is breaking the protocol's invariant, as specified in the docs:

```
Invariant with fees
ρ = fee (between 0 & 1, aka a percentage)
γ = (1 - p) (pronounced gamma)
∆x = (β/(1-β)) * (1/γ) * x
∆y = (αγ/1+αγ) * y
```

**Impact:**

This will lead to protocol insolvency.

**Proof Of Concept:**

Check `test/Invariant.t.sol`.

**Recommended Mitigation:**

Remove this feature or use the fees eventually generated by the protocol.
